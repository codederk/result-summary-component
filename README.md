### What I learned

I found out how to put/embed svg images in HTML by using <object> element.

```html
<object data="./assets/images/icon-reaction.svg"></object>
```

I also learned about Media Queries,

```css
@media screen and (min-width: 756px) {
  p {
    font-size: 18px;
  }
}
```
